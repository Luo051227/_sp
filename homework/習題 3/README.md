# 密碼產生器 - 專案說明

## 專案概述

這是一個功能完整的密碼產生器專案，提供圖形介面（GUI）和命令列介面（CLI）兩種版本。

## 功能特色

### 圖形介面版本 (密碼產生器.py)
- ✅ 產生隨機安全密碼
- ✅ 自訂密碼長度（4-64字元）
- ✅ 選擇包含的字元類型（大寫、小寫、數字、特殊符號）
- ✅ 排除容易混淆的字元
- ✅ 產生易記的片語密碼
- ✅ 密碼強度評估
- ✅ 一鍵複製到剪貼簿
- ✅ 密碼歷史記錄

### 命令列版本 (密碼產生器CLI.py)
- ✅ 完整參數控制
- ✅ 批次產生多個密碼
- ✅ 易記片語模式
- ✅ 安靜模式（適合腳本使用）

## 執行方式

### GUI 版本
```bash
python3 密碼產生器.py
```

### CLI 版本
```bash
# 產生預設密碼（16字元）
python3 密碼產生器CLI.py

# 產生20字元密碼
python3 密碼產生器CLI.py -l 20

# 產生包含所有字元的12字元密碼
python3 密碼產生器CLI.py -l 12 -u -d -p

# 一次產生5個密碼
python3 密碼產生器CLI.py -n 5

# 產生易記片語
python3 密碼產生器CLI.py --memorable

# 安靜模式
python3 密碼產生器CLI.py -q
```

## 參數說明

| 參數 | 說明 |
|------|------|
| `-l, --length` | 密碼長度 (預設: 16) |
| `-u, --uppercase` | 包含大寫字母 |
| `-d, --digits` | 包含數字 |
| `-p, --punctuation` | 包含特殊符號 |
| `-e, --exclude-ambiguous` | 排除容易混淆的字元 |
| `-n, --count` | 一次產生的數量 |
| `-m, --memorable` | 產生易記片語 |
| `-q, --quiet` | 安靜模式 |

## 密碼強度說明

| 評分 | 等級 | 說明 |
|------|------|------|
| ★☆☆☆☆ | 非常弱 | 短於8字元，單一字元類型 |
| ★★☆☆☆ | 弱 | 8-11字元，1-2種字元類型 |
| ★★★☆☆ | 中等 | 12-15字元，2-3種字元類型 |
| ★★★★☆ | 強 | 12-15字元，4種字元類型 |
| ★★★★★ | 非常強 | 16+字元，4種字元類型 |

## 技術說明

- **GUI**: 使用 Python 內建的 tkinter 模組
- **隨機數**: 使用 Python 的 random 模組
- **字元集**: 使用 Python 的 string 模組

## 環境需求

- Python 3.6+
- 無需安裝額外套件
